Everything you need to know about running Windows in your browser with PC-Free.

First boot   Takes 5 – 10 minutes while downloading the Windows image from Microsoft's servers. This only happens once.

Subsequent boots   Once cached, your environment is up in 2 – 3 minutes.
Free tier   Yes! GitHub provides 120 hours/month of Codespaces for free. This is enough for light daily use.

Paid plans   If you need more power or longer sessions, upgrade to a paid Codespaces plan — PC-Free just runs on top of it.
Yes — fully. PC-Free exposes a real RDP (Remote Desktop) session, so your keyboard layout, shortcuts, and peripherals work exactly as they would on a local Windows machine.
Yes. Since everything runs in the browser, any device with a modern browser works — including Android, iPad, and ChromeOS. For the best experience, use landscape mode.
Your files live on GitHub's infrastructure — not on our servers. We never see, store, or touch your data. GitHub Codespaces sessions can be encrypted at rest and in transit.
PC-Free is just a web interface. It runs entirely in the browser. The "heavy lifting" (Windows, Docker, RDP) happens inside a GitHub Codespace running Ubuntu.
